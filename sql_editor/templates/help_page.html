SQL Help Page

1. How do I write a SELECT query?
To retrieve data from a table, you can use a SELECT statement. For example: 
SELECT * FROM your_table_name;
2. How can I filter data in a SELECT query?
You can use the WHERE clause to filter data in a SELECT query. For example: 
SELECT * FROM your_table WHERE column_name = 'value';
3. How do I create a new table in SQL?
To create a new table, you can use the CREATE TABLE statement. For example: 
CREATE TABLE new_table_name (
    column1 datatype,
    column2 datatype,
    ...
);
4. How can I perform basic math operations in SQL queries?
SQL supports basic mathematical operations such as addition (+), subtraction (-), multiplication (*), and division (/). For example: 
SELECT column1 + column2 AS sum_result FROM your_table;
SELECT column1 - column2 AS difference_result FROM your_table;
SELECT column1 * column2 AS product_result FROM your_table;
SELECT column1 / column2 AS division_result FROM your_table;
5. How do I calculate the average (mean) of a column?
You can use the AVG() function to calculate the average of a column's values. For example: 
SELECT AVG(column_name) AS average_value FROM your_table;
6. How can I find the maximum and minimum values in a column?
To find the maximum and minimum values in a column, use the MAX() and MIN() functions, respectively. For example: 
SELECT MAX(column_name) AS max_value FROM your_table;
SELECT MIN(column_name) AS min_value FROM your_table;
7. How do I concatenate two strings in SQL?
You can concatenate two strings in SQL using the CONCAT() function or the `||` operator. Here's an example with CONCAT(): 
SELECT CONCAT(first_name, ' ', last_name) AS full_name FROM your_table;
8. How can I count the number of rows in a table?
To count the number of rows in a table, you can use the COUNT() function. For example: 
SELECT COUNT(*) AS row_count FROM your_table;
9. How do I convert a string to uppercase in SQL?
To convert a string to uppercase in SQL, you can use the UPPER() function. For example: 
SELECT UPPER(column_name) AS uppercase_string FROM your_table;
10. What is the purpose of the LENGTH function in SQL?
The LENGTH function in SQL is used to determine the length of a string, which is the number of characters in the string. For example: 
SELECT LENGTH(column_name) AS string_length FROM your_table;
